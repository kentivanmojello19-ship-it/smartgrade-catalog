# SmartGrade Solutions | Premium Order Manager

**High Performance. Minor Flaws. Major Savings.**

SmartGrade Solutions is a premium, aesthetic, and functional web-based catalog. This tool allows you to manage industrial product inventory, track customer orders with quantity counts, and generate professional 3D-styled receipts.

---

## ✨ Features
* **💎 Premium 3D Aesthetic:** Glassmorphism UI and floating product cards.
* **🔍 Live Search:** Instant filtering to find specific industrial products.
* **🛒 Order System:** Adjust pieces (pcs) and calculate totals automatically.
* **💾 Save to File:** Generates a professional **Receipt Image (PNG)** to send to customers.

---

## 🛠 Usage
1.  **Add Item:** Upload photos and set prices for "reject" or "off-grade" items.
2.  **Order Now:** Click to add an item to the cart.
3.  **Adjust Qty:** Use the + or - buttons to change the number of pieces.
4.  **Save Receipt:** Click "Save Receipt Image" to download the final order flyer.
